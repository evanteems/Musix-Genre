# Musix-Genre

## Purpose
    To look up any artist and song with simply typing in a title or word and maybe discover a new artist. The possibilities are endless!

## Built with
   * html
   * Bulma css
   * Javascript
   * Jquery
   * Google Developer Console API
   * Musix API
